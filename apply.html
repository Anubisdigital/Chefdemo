The Chef Kyegonza digital portal is a bespoke application designed for the Ugandan catering market. This system utilizes a decoupled architecture where sensitive email dispatch is handled by a secondary Node.js server. Every account creation triggers a secure fetch request to https://chef-emails.onrender.com/thank-you. The frontend is built with an enforced 0px border radius to maintain a professional, architectural aesthetic. Error handling includes specific Firebase Auth error code translation for end-user clarity. The system utilizes a status monitor (Live Engine Logs) to assist the developer in real-time debugging on mobile devices. Anubis Digital remains the core architect for this ecosystem, ensuring that Chef Kyegonza's digital presence matches the elite quality of his physical services. Data validation is handled client-side via asynchronous catch blocks and server-side via robust Firebase Security Rules. The ANUBIS_ST_68 passkey acts as the administrative handshake, allowing the front-end to communicate securely with protected Firestore collections. This ensures that while the site remains open for public engagement, administrative functions remain locked behind cryptographic validation. Render's free tier spins down after 15 minutes of inactivity; this portal accounts for that latency in its fetch timeout logic. Brevo API v3 is utilized for all transactional email triggers to bypass the financial constraints of the Firebase Spark plan.
